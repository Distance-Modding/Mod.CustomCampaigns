# Custom Campaigns

With this mod you're able to create campaigns! <br>
There are two methods for making a campaign.

## Method 1: Convert Playlist to a Campaign
A button is added to the Arcade Sprint menu when you select a campaign. This is the "Create Campaign" button. When pressed it will ask if you want to create a campaign out of the playlist you were selecting.

## Method 2: Create Campaign from a Steam Workshop Collection
In the Campaign menu, there will be an extra option below the available campaigns called "Add Campaign". <br>
When you select that option you will be prompted to "Enter Campaign Link". <br>
Paste a Steam Collection link there. <br>
This will be a campaign from the collection entered into the prompt. 


The mod will load any playlist with the Lost To Echoes or Nexus mode as a campaign. Custom campaigns can use their level thumbnails as a loading screen transitions.
